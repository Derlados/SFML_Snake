Для запуска проекта необходимо скачать графическую библиотеку с сайт https://www.sfml-dev.org/

Настройка в Visual Studio:
1. Зайдите в свойства проекта
2. В разделе Свойства конфигурации/"C/C++"/Общие в строке "Дополнительные катологи катологи включаемых файлов" указать
   полный путь к директории include которая которя находится в директории библиотеки SFML к примеру C:\SFML-2.5.1\include
3. В разделе Компоновщик так же само указать путь к директории lib - C:\SFML-2.5.1\lib
